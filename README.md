# MyLinkSonar

自社サイトに掲載されたリンク切れ・リソース不備を、安全かつ低負荷で検出するデスクトップツールです。 

## ダウンロード

[最新リリース](https://github.com/boxtriq/MyLinkSonar/releases/latest) から `MyLinkSonar.zip` をダウンロードしてください。 

## 特徴

- 外部通信なし（実行時）― チェック処理はすべてローカル PC 上で完結
- 低速・低負荷設計（デフォルト 5 秒間隔・並列数 1）
- ブロック検知時の自動停止（403 / 429）
- クロールモード・サイトマップモードの 2 モード対応
- CSV 自動保存 ＋ クリップボード共有（Slack・Notion 等への貼り付け用）
- オフラインライセンス対応
- インストール不要（ポータブルアプリ）

## 動作環境

- Windows 64bit

## 使い方

1. [リリースページ](https://github.com/boxtriq/MyLinkSonar/releases/latest) から `MyLinkSonar.zip` をダウンロード
2. ZIP を展開し、`MyLinkSonar 1.0.0.exe` を起動
3. プロファイルを作成して実行

詳細は [クイックガイド](https://boxtriq.com/mylinksonar/help/quick-guide.html) をご参照ください。

## ライセンス

MyLinkSonar は有償ライセンスのソフトウェアです。

|| 未ライセンス | ライセンス済み |
|--|------------|--------------|
| 最大ページ数 | 60 | 10,000 |
| 最大深度 | 1 | 2 |
| 1 ページあたりリンク上限 | 100 | 500 |
| 実行回数 | 合計 3 回（以降 1 日 1 回） | 無制限 |

ライセンスの購入は [製品サイト](https://boxtriq.com/mylinksonar/) から。
